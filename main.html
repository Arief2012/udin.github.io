About me

Saya seorang mahasiswa Informatika di Universitas Amikom Purwokerto, dan saya sangat tertarik dengan teknologi dan pemrograman. Sejak kecil, saya sudah memiliki ketertarikan pada komputer dan pengembangan software, dan kini saya menjalani passion saya melalui pendidikan di jurusan Informatika.
Selama kuliah, saya banyak belajar tentang pemrograman, database, jaringan komputer, keamanan informasi, dan teknologi terkini lainnya. Saya juga mengambil Studi Independen di Power Academy untuk meningkatkan kemampuan dan keterampilan saya dalam bidang teknologi khususnya kecerdasan buatan atau AI.
Saya memiliki motivasi yang tinggi untuk terus belajar dan mengembangkan diri dalam bidang teknologi. Saya ingin menjadi seorang profesional di bidang teknologi yang dapat memberikan dampak positif bagi masyarakat.

Portofolio

Saya membuat sebuah aplikasi mobile dengan bahasa Java dan sofware android studio untuk membantu pengguna dalam mencatat kegiatan atau peristiwa dalam bentuk diary atau catatan harian. Aplikasi ini dilengkapi dengan fitur login, register, Tambah catatan, Update catatan, hapus catatan dan penggunaan Google Ads untuk memasang iklan.

Saya membuat sebuah proyek 3D isometric ruangan dengan teknik rendering isometrik menggunakan software Blender. Proyek ini melibatkan pembuatan model 3D, pengaturan pencahayaan, dan rendering untuk menghasilkan gambar realistis.

Saya membuat aplikasi inventory untuk Kecamatan Karang Lewas menggunakan teknologi .NET, aplikasi Visual Studio, dan database SQL Server. Aplikasi ini membantu pengelolaan inventaris barang dan aset dengan mudah. Aplikasi ini terintegrasi dengan database SQL Server dengan baik. Proyek ini menunjukkan kemampuan saya dalam pengembangan aplikasi desktop dengan teknologi .NET dan aplikasi Visual Studio serta penggunaan database SQL Server.
